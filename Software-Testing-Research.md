**Software Testing (Manual + Automation Testing with Selenium \& Java)**



Software Testing is the process of verifying and validating a software application to ensure that

it meets business requirements and works correctly without defects. It helps improve software quality,

reliability, and performance.



Software testing is mainly divided into:



**Manual Testing**



Testing performed by a tester without using automation tools.

Test cases are executed manually.

Used for exploratory, usability, and ad-hoc testing.



**Automation Testing**



Testing performed using tools and scripts.

Reduces repetitive manual work.

Faster and more reliable for regression testing.
